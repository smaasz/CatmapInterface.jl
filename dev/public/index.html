Interface

CatmapInterface.parse_catmap_input — Function
parse_catmap_input(input_file_path)

Parse the content of a CatMAP input file needed for a Poisson-Nernst-Planck model.

The input file is first interpreted by a Python interpreter and then parsed. The following information is used and must be specified:

  • rxn_expressions
  • prefactor_list
  • species_definitions (including ...)
  • input_file (including the energy table)
  • surface_names
  • bulk_ph
  • Upzc
  • potentialreferencescale
  • gasthermomode
  • adsorbatethermomode
  • electrochemicalthermomode

See CatMAP documentation for details.

CatmapInterface.CatmapParams — Type
struct CatmapParams

Main data structure for interfacing CatMAP

  • reactions::Vector{CatmapInterface.ParsedReaction}: List of parsed reactions
  • prefactors::Vector{Float64}: Prefactors in the Arrhenius relation specifying the activation energy of the corresponding reactions
  • species_list::Dict{String, AbstractSpecies}: List of species including all reactants
  • gas_thermo_mode::Symbol: Mode of the thermodynamical correction to the formation energy of the gases
  • adsorbate_thermo_mode::Symbol: Mode of the thermodynamical correction to the formation energy of the adsorbates and transition states
  • electrochemical_thermo_mode::Symbol: Mode of the electrochemical correction to the formation energy of the species
  • bulk_pH::Float64: pH value in the bulk of the electrolyte
  • Upzc::Float64: Potential of zero charge with respect to the specified potential reference scale
  • potential_reference_scale::String: Either RHE or SHE
  • T::Float64: Temperature in the bulk

Reaction Network

CatmapInterface.create_reaction_network — Function
create_reaction_network(catmap_params)

Create a ReactionSystem for the specified microkinetic model.

For each elementary reaction the CatmapInterface.ratelaw_TS is used. No separate rate equations for the solvent (= H₂O) and the active sites are created but their activities can be specified as parameters. For ficitious gases (OH⁻ and H⁺) and adsorbates the activity coefficients are assumed to be 1. The activity coefficients of the gaseous species can specified as parameters. The thermodynamical corrections to the DFT-data of the formation energies are applied according to the specified modes. New modes can be added by the user by adding a function with the same name to the module.
